# @omnitron-dev/titan-discovery

> Redis-backed service discovery for the Titan framework

Part of the [Omni](../../README.md) monorepo — Fullstack Type-Safe RPC Framework.

## Installation

```bash
pnpm add @omnitron-dev/titan-discovery
```

## Overview

Enables Titan services to register themselves and discover other services in a distributed environment. Uses Redis as the backing store for service registration, heartbeats, and lookup.
