# Layout Primitives

Layout and spacing components for building responsive interfaces.

## Containers

- **[Box.md](./Box.md)** - Universal layout box with all CSS properties
- **[Container.md](./Container.md)** - Responsive content containers with max-width
- **[Flex.md](./Flex.md)** - Flexbox layouts with shorthand props
- **[Grid.md](./Grid.md)** - CSS Grid layouts with template areas
- **[Stack.md](./Stack.md)** - Vertical/horizontal stacks with spacing
- **[SimpleGrid.md](./SimpleGrid.md)** - Responsive grid with equal columns
- **[Masonry.md](./Masonry.md)** - Masonry grid layouts (Pinterest-style)

## Spacing & Alignment

- **[Space.md](./Space.md)** - Flexible spacing between child elements
- **[Spacer.md](./Spacer.md)** - Flex-grow based spacers
- **[Center.md](./Center.md)** - Center content vertically/horizontally
- **[Divider.md](./Divider.md)** - Visual content separators with labels
- **[Separator.md](./Separator.md)** - Semantic ARIA separators

## Scrolling & Ratios

- **[ScrollArea.md](./ScrollArea.md)** - Custom scrollable areas with styled scrollbars
- **[AspectRatio.md](./AspectRatio.md)** - Maintain aspect ratios (16:9, 4:3, etc.)

## Common Features

- Responsive design with breakpoints
- CSS-in-JS with style props
- Gap and spacing utilities
- Alignment and justification controls
- Direction and orientation support
- Nested layouts and composition
