# @omnitron-dev/titan-ratelimit

> Redis-backed rate limiting for the Titan framework

Part of the [Omni](../../README.md) monorepo — Fullstack Type-Safe RPC Framework.

## Installation

```bash
pnpm add @omnitron-dev/titan-ratelimit
```

## Overview

Rate limiting module backed by Redis. Supports fixed-window and sliding-window algorithms with configurable limits per key, endpoint, or user.
