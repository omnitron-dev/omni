# @omnitron-dev/titan-telemetry-relay

> Store-and-forward telemetry pipeline for distributed Titan applications

Part of the [Omni](../../README.md) monorepo — Fullstack Type-Safe RPC Framework.

## Installation

```bash
pnpm add @omnitron-dev/titan-telemetry-relay
```

## Overview

Collects telemetry data (traces, metrics, logs) from distributed Titan services and forwards it to configured backends. Uses a store-and-forward pattern to handle network interruptions without data loss.
